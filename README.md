# Códigos del Pregrado de Física

El presente repositorio alberga varios códigos desarrollados a lo largo de mi pregrado de Física en la Universidad Nacional de Colombia. Están organizados por cursos, salvo algunos notebooks sueltos que preferí no agrupar en carpetas individuales debido a la poca relación entre sí; sin embargo, no son menos valiosos. El propósito de conservar estos programas es meramente personal, mas no privado.
